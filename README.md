# Python基于PyQt5的图书管理系统 python744

## 项目概述

本项目是一款基于Python和PyQt5开发的图书管理系统。系统提供图书信息与读者信息的维护，具备增删改查等基本功能，并支持从文件导入信息，特别适合新手学习和使用。

## 技术栈

- Python
- PyQt5

## 运行环境

- Python 3.x
- PyQt5

## 功能模块

### 图书管理

- 图书信息录入
- 图书信息修改
- 图书信息删除
- 图书信息查询

### 读者信息维护

- 读者信息录入
- 读者信息修改
- 读者信息删除
- 读者信息查询

### 数据处理

- 支持从文件导入图书与读者信息

## 系统角色

- 系统管理员：负责图书与读者信息的管理和维护

## 目录结构

```plaintext
图书管理系统/
├── main.py               # 程序主入口
├── forms/                # 界面相关文件
│   ├── main_window.ui    # 主窗口UI文件
│   └── ...
├── models/               # 数据模型相关文件
│   ├── book.py           # 图书数据模型
│   └── reader.py         # 读者数据模型
├── views/                # 视图层文件
│   ├── main_window.py    # 主窗口逻辑
│   └── ...
└── controllers/          # 控制器相关文件
    └── main_controller.py # 主要逻辑控制
```

- 使用Python语言和PyQt5框架，易于理解和学习
- 清晰的目录结构，便于项目维护和扩展
